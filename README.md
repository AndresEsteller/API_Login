**Descripción de la API**

La API de Login proporciona un sistema completo de autenticación y gestión de usuarios. Incluye funcionalidades esenciales para el control de acceso y administración de cuentas, tanto para usuarios normales como con roles asignados.

**--- Funcionalidades principales ---**

 - Registro de usuarios (con y sin rol asignado).
 - Inicio y cierre de sesión.
 - Restablecimiento de contraseña mediante la opción "Olvidé mi contraseña".
 - Cambio de contraseña desde la cuenta del usuario.
 - Gestión de roles y estados de usuarios (activar, desactivar, actualizar rol).
 - Consulta de intentos de inicio de sesión.
 - Obtención de la lista de usuarios registrados.
